Secure donation/payment direction

Use trusted third-party platforms.

I would not build custom payment collection directly into the website. The safer approach is to use Zeffy, Square, PayPal, or Stripe.

Zeffy is highlighted first because it is built for nonprofits and can help keep more donation dollars pointed toward the mission, subject to eligibility and proper organization setup.

Suggested payment options

  • Zeffy: preferred nonprofit donation/fundraising option to review first.
  • Square: simple and familiar for payments/events.
  • PayPal: widely recognized by donors.
  • Stripe: flexible for custom payment flows later.

No card data should be stored on this website.
